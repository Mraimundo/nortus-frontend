# Internacionalização (i18n)

Arquitetura:
- Rotas localizadas: src/app/[locale]/...
- Utilitários: src/i18n/routing.ts, src/i18n/request.ts
- Mensagens: messages/pt-BR.json, messages/en-US.json

Padrão de uso:
- Obter locale a partir da rota.
- Carregar mensagens do respectivo arquivo em messages/.
- Consumir textos via hooks/utilitários de i18n conforme implementados.

Adicionando um novo idioma:
- Criar messages/<locale>.json
- Atualizar src/i18n/routing.ts com o locale suportado.
- Garantir páginas em src/app/[locale] utilizam o locale dinamicamente.
